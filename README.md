# ITCShop - Admin Panel

This is the **Admin Panel** for ITCShop, part of a full-stack e-commerce platform developed during the Software Engineering program at the **Institute of Technology of Cambodia (ITC)**.

The Admin Panel allows administrators to manage products, categories, orders, users, and monitor key store analytics. It connects to the ITCShop Laravel backend API.

---

## ✨ Features

- Admin authentication (login/logout)
- Product and category management (CRUD)
- View and manage customer orders
- Dashboard with analytics and revenue stats
- Manage admin/staff accounts
- Customer overview and coupon management

---

## 🧩 Admin API Integration

| Controller           | Purpose                                                                  |
|----------------------|--------------------------------------------------------------------------|
| AuthController       | Admin login and logout                                                   |
| DashboardController  | Show analytics, order stats, and total revenue                           |
| CategoryController   | CRUD operations for product categories                                   |
| ProductController    | CRUD operations for products                                             |
| OrderController      | View and update order statuses                                           |
| UserController       | Manage admin and staff users                                             |
| CustomerController   | View and manage customer accounts                                        |
| CouponController     | Create, update, and delete coupons                                       |

---

## 🛠 Tech Stack

- **Vue 3**
- **Vite**
- **Pinia** (State Management)
- **Vue Router**
- **Axios** (API Communication)
- **Tailwind CSS** (UI Framework)
